- 마크다운 사용법 (.md)
- 깃 사용법(계정), 내 폴더관리 
 - git add 선택 - 선택(생성,수정,삭제)
 - git commit-m "메모" - 메모
 - git push - 보내기 
 - git pull - 받기
 - 만약, 컴퓨터에 기존 git 자료가 없다면 git clone 주소로 자료 동기화(받기)
